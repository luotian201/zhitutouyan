🚀 快速开始（3步部署）
═══════════════════════════════════════════════════════════

第一步：在 GitHub 创建仓库
--------------------------
1. 访问 https://github.com 并登录
2. 点击右上角 "+" → "New repository"
3. 仓库名称：zhitu-invest-website（可自定义）
4. 选择 Public
5. 点击 "Create repository"
6. 复制仓库地址（类似：https://github.com/用户名/zhitu-invest-website.git）


第二步：推送代码到 GitHub
--------------------------
打开命令行（CMD 或 PowerShell），执行：

cd C:\Users\tnk2tnk\Desktop\zhitutouyan\zhitu-invest

git remote add origin https://github.com/您的用户名/zhitu-invest-website.git

git branch -M main

git push -u origin main

（替换上面的仓库地址为您自己的）


第三步：部署到 Vercel
--------------------------
1. 访问 https://vercel.com
2. 用 GitHub 账号登录
3. 点击 "Add New..." → "Project"
4. 选择 zhitu-invest-website 仓库
5. 点击 "Deploy"
6. 等待 1-2 分钟，完成！

🔧 常见问题
═══════════════════════════════════════════════════════════

Q: 推送到 GitHub 需要密码？
A: GitHub 现在使用 Personal Access Token
   Settings → Developer settings → Personal access tokens → 生成新 token

Q: Vercel 部署失败？
A: 检查 GitHub 代码是否完整，查看 Vercel 构建日志

Q: 如何修改样式？
A: 修改组件中的 Tailwind CSS 类名，或编辑 app/globals.css

Q: 如何添加新页面？
A: 创建新组件，然后在 app/page.tsx 中导入
